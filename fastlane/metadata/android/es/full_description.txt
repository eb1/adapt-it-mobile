<p><i>Adapt It Mobile</i> es una herramienta sencilla para la traducción entre idiomas similares diseñada para dispositivos móviles. Con una tableta o teléfono inteligente, puede usar Adapt It Mobile para traducir rápidamente el texto de un idioma a otro similar. Adapt It Mobile recuerda las traducciones anteriores y se sugieren la próxima vez que se encuentre la palabra o frase.</p><p><i>Adapt It Mobile</i> puede importar y exportar lo siguiente desde el portapapeles o desde un archivo:</p><ul><li>.usfm</li><li>.sfm</li><li>.usx</li><li>Documentos de Adapt It (.xml)</li><li>Texto sin formato (.txt)</li><li>Datos de la memoria de traducción: <code>Archivos de la base de conocimientos de Adapt It (.xml)</code><code>Intercambio de memoria de traducción (.tmx)</code>Términos clave en formato SFM (.sfm)<code></code><code>Formato de intercambio léxico (.lift)</code></li></ul><p>La interfaz de usuario de Adapt It Mobile está localizada para inglés, español, portugués, francés y tok pisin.</p>
