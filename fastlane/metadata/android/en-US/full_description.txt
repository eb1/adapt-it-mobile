<p><i>Adapt It Mobile</i> is a simple related language translation editor designed for mobile devices. Using a tablet or smartphone, you can use Adapt It Mobile to quickly translate text from one language to another related language. Previous translations are stored in a translation memory called a Knowledge Base, and are suggested the next time that particular word or phrase is encountered.</p><p><i>Adapt It Mobile</i> can import and export the following:</p><ul><li>Clipboard text</li><li>USFM markup files: <code>.usfm</code>, <code>.sfm</code>, <code>.usx</code></li><li>Adapt It (desktop) documents: <code>.xml</code></li><li>Plain-text files: <code>.txt</code></li></ul>
