<p><i>Adapt It Mobile</i> is a simple related language translation editor designed for mobile devices. Using a tablet or smartphone, you can use Adapt It Mobile to quickly translate text from one language to another related language. Previous translations are stored in a translation memory called a Knowledge Base, and are suggested the next time that particular word or phrase is encountered.</p><p><i>Adapt It Mobile</i> can import and export the following from the clipboard or from a file:</p><ul><li>.usfm</li><li>.sfm</li><li>.usx</li><li>Adapt It (desktop) documents (.xml)</li><li>Plain-text (.txt)</li><li>Translation memory data: <code>Adapt It Knowledge Base files (.xml)</code><code>Translation Memory Exchange (.tmx)</code>Key Terms in SFM format (.sfm)<code></code><code>Lexical Interchange Format (.lift)</code></li></ul><p>Adapt It Mobile's user interface is localized for English, Spanish, Portuguese, French, and Tok Pisin.</p>
